### BUG-022: Отсутствие плавающей кнопки CTA на мобильных
**Файл:** `src/App.jsx` / `src/components/FloatingCTA.jsx`
**Статус:** В ожидании.
**Проблема:** На узких экранах кнопка "Заказать расчет" уходит из вида. Нужна фиксированная кнопка внизу экрана.

### BUG-025: Некорректный курсив (Faux Italic) в заголовке "Под ключ"
**Файл:** `src/components/Guarantee.css` (`.guarantee__accent`)
**Статус:** В ожидании.
**Проблема:** Использование `font-style: italic` для кириллицы гротескных шрифтов создает искаженный наклон (faux italic). Выглядит дешево и ломает ритм.
**Решение:** Убрать курсив. Оставить акцентный цвет и, возможно, добавить небольшое подчеркивание (underline) или сделать фон в виде штриха маркера под текстом.

### BUG-026: Удалить временные иконки-эмодзи в футере
**Файл:** `src/components/Footer.jsx` (`.footer__social`)
**Статус:** В ожидании.
**Проблема:** Иконки Telegram (💬) и WhatsApp (📱) в виде эмодзи выглядят непрофессионально. Пользователь просит их убрать совсем.
**Решение:** Удалить блок `.footer__social` из компонента `Footer.jsx`.
